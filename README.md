# Amet Alvirde

This is the source code for my personal website: [AmetAlvirde.com][1]

[1]: https://www.AmetAlvirde.com

## Styles

The `src/styles/global.css` file shares the Margarita color token/semantic layer with the astro-seed project. When Margarita colors change, update both `amet-alvirde/src/styles/global.css` and `astro-seed/src/styles/global.css` to keep them in sync. Site-specific component styles (cta-button, wayfinding, etc.) live in `src/styles/site.css`.
